## How to Use These Config Files

Each config file comes with **embedded instructions and comments**. 
- Go to each file, even the no-need-to-edit ones, And see where to paste them, How to create the folders or files. 
- Open the SwayWM config, It isn't plug and play- You need to add some manual entries, As some apps are different depending on how you use the computer. Go to the sway folder for this.
- Open Waybar config and CSS to customize the status bar.- But you can use the default ones i gave, for a "vanilla" look
- Open Wofi CSS to tweak your app launcher and emoji menu.- Same as the waybar one, You can paste the default ones.

Follow the comments inside each file—they guide you step by step.

This should be quiet simple- Expect you as the user to finish setting it up under 20 minutes, Feel free to set it up from a DE like GNOME/KDE/LXQt, Then log into swaywm.

Have a enjoyable experience!
